# Sample Audio Files

This folder contains sample audio files generated by the qtts CLI to demonstrate its capabilities.

## Generated Samples

### test_ryan.mp3
- **Text:** "Hello world, this is a test of the Qwen3 TTS system!"
- **Mode:** Custom Voice
- **Speaker:** Ryan (Dynamic male, strong rhythmic drive - English native)
- **Language:** English
- **Command:** `./qtts.py "Hello world, this is a test of the Qwen3 TTS system!" -s Ryan -l English -o test_ryan.mp3`

### test_serena_french.mp3
- **Text:** "Bonjour! Comment allez-vous?"
- **Mode:** Custom Voice
- **Speaker:** Serena (Warm, gentle young female - Chinese native, speaking French)
- **Language:** French
- **Command:** `./qtts.py "Bonjour! Comment allez-vous?" -s Serena -l French -o test_serena_french.mp3`

### test_voice_design.mp3
- **Text:** "Welcome to the future of text to speech!"
- **Mode:** Voice Design
- **Voice Description:** "Energetic female voice, 25 years old, enthusiastic tech presenter style"
- **Language:** English
- **Command:** `./qtts.py "Welcome to the future of text to speech!" -m design -i "Energetic female voice, 25 years old, enthusiastic tech presenter style" -l English -o test_voice_design.mp3`

## Notes

- All samples were generated using the Qwen3-TTS models running on CPU (macOS)
- Generation time: ~2-5 seconds per sample on CPU
- Audio format: MP3 (converted from WAV via ffmpeg)
- Sample rate: 24kHz

## Regenerating Samples

You can regenerate these samples or create new ones using the commands above. Make sure you have:
1. Installed all dependencies (`pip install -r requirements.txt`)
2. Installed ffmpeg (`brew install ffmpeg`)
3. Activated the virtual environment (`source venv/bin/activate`)
